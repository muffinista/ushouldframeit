You Should Frame It!

This is a bot that you can add to your slack channel. It will add frames to pictures for you!

Add it to Slack

How Does It Work?

After you add the bot to a Slack channel, you can send it commands like this:

@you_should_frame_it: frame [IMAGE URL]

And it will take that image and put it in a frame.
